# 导航与阅读路径
说明展示页、知识库、资产索引和生产文档之间如何跳转与分工。
Route Map
## 新版入口分工
新版允许跳转和拆页，但每个入口要有清楚职责。
### 展示入口
从首页和两张角色页快速看到项目视觉、角色设定页和资产槽位。适合浏览和检查网页观感。
### 知识库入口
从知识库总目录进入导航、人设、故事、视觉提示词、制作规范。适合阅读和扩写内容。
### 资产入口
从资产索引页集中查看角色根目录、截图、提示词、计划输出和登记状态。
### 生产入口
资产生成、并行分工、坑点记忆和资产登记走 workflows/、tools/ 和 logs/，不塞回展示页。
