# Navigation And Reading Paths
Explains how showcase pages, knowledge pages, the asset index, and production docs connect.
Route Map
## New Entry Responsibilities
The new site can jump across pages and split content, but every entry must have a clear job.
### Showcase Entry
Use the index and character sheets to inspect visual presentation, character sheets, and asset slots.
### Knowledge Entry
Use the knowledge index for navigation, characters, story, visual prompts, production rules, and asset planning.
### Asset Entry
Use the asset index to inspect character roots, crop handles, prompt notes, planned outputs, and registry status.
### Production Entry
Asset generation, parallel ownership, issue memory, and asset registry live in workflows, tools, and logs.
