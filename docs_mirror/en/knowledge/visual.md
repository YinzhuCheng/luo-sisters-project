# Visual Direction And Prompt Library
Art direction, post-layout text rules, expression rules, duo key visual, and prompt groups for both sisters.
Art Direction
## Unified Art Direction
Images serve character identity and reproducibility first; final text belongs to web or post-layout tools.
### Style
Japanese anime character sheet, clean lineart, soft cel shading, clear detail, low-saturation colors, light-novel cover and production-reference mood.
### Image Control
Readable outfit construction, reusable accessories, stable hair silhouettes, and clear color roles.
### Text Rule
Generate characters, outfits, ornaments, and blank layout space first. Add titles, labels, and long copy through HTML, SVG, or PIL.
### Expression Rule
Every expression must visibly change brows, eyes, mouth, head angle, gesture, or shoulders.
