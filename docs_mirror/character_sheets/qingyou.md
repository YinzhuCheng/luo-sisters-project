- 提前到达约定地点，包里常备发夹、胶带、便签和创可贴。
- 拍照前先看光，再看人。
- 喜欢给灵感留一页，也喜欢给计划留一栏“意外”。
- 听见有栖的新点子，会先翻开手账，再微笑回应。
