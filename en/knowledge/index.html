Structure Rules

Structure Rules

Showcase And Knowledge Separation

The index and character sheets stay lightweight, attractive, and responsive. Knowledge pages carry character, story, prompt, workflow, and asset-planning depth. The asset index owns path discovery.

Build Structure First

Stabilize entrypoints, asset paths, and workflows first. Then deepen story, persona, and prompt detail page by page.

Follow Stable Entry Points

Each knowledge page points toward character sheets, the asset index, or workflows. When you need more detail, continue one level deeper instead of scanning the whole site.
