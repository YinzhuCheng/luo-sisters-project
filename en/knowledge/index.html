Migration Rule

Migration Rules

Showcase And Knowledge Separation

The index and character sheets stay lightweight, attractive, and responsive. Knowledge pages carry character, story, prompt, workflow, and asset-planning depth. The old archive stays intact as the source record.

Migration Is Not Verbatim Copying

The first pass builds full navigation, summaries, and source anchors. Later passes can expand individual pages without losing the original archive.

Prevent False Content Loss

Each knowledge page shows structured content, migration status, and old archive anchors. If something looks missing, check knowledge pages, then the old archive, then docs/content_map.md.
